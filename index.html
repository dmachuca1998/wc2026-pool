📋 How It Works

1. Admin adds all participants in Setup.
2. Select each person from the dropdown on the Predictions tab and enter their predicted scores for all 72 group stage matches.
3. On the Bracket tab, each person picks which 8 third-place groups qualify, then predicts winners (by group position) through every knockout round to the Final.
4. On the Bonus tab, enter each person's bonus picks (champion, golden boot, total goals).
5. Once the tournament starts, the Admin enters real results on the Admin tab.
6. The Leaderboard auto-calculates all scores in real time!
7. Use the Data tab to export/import all data (backup frequently!).
